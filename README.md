# 📊 Habit Tracker

## 📌 Overview
Habit Tracker is a web-based application designed to help users build and maintain daily habits. It allows users to track progress, maintain streaks, and stay consistent using reminders and progress tracking.

---

## 🚀 Features

### 🔹 Core Features
- Add new habits
- Display list of habits
- Mark habits as completed for today
- Store habit data

### 🔹 Advanced Features
- Streak tracking system
- Weekly completion percentage
- Reminder messages for pending habits
- Data persistence using Local Storage

### 🌟 Creative Feature
- Streak Badge System (Bronze, Silver, Gold)
- Progress visualization for habits

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Browser Local Storage


---

## ⚙️ How It Works
1. User adds a new habit.
2. Habit is displayed in the list.
3. User marks habit as completed daily.
4. System:
   - Updates streak
   - Calculates weekly progress
5. Data is saved in local storage and persists after refresh.

---

## ▶️ How to Run
1. Download or clone the repository
2. Open `index.html` in any browser
3. Add habits and track progress
